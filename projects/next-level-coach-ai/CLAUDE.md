# CLAUDE.md — Next Level Coach AI

## What This Project Is
Next Level Coach AI is a specialized AI software platform built specifically for online coaches. It provides purpose-built AI agents that assist with email communication, client retention, lead follow-up, content strategy, and knowledge scaling.

## Founders
Built by Brett and Samantha. Brett handles the technical side. Samantha is the coach and product vision.

## Before Working On This Project
Always read the context folder files before making decisions. Start with:
1. context/project-brain-entry-point.md — navigation guide
2. context/master-overview.md — what the company is
3. context/product-current-state.md — what currently exists
